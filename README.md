# ✨ Overview

AI Chatbot UI is a responsive and interactive frontend chatbot application developed using ReactJS.  
The project provides a modern conversational interface where users can interact with an AI-style chatbot in real time.

This project demonstrates:
- Component-based architecture
- Dynamic rendering
- State management using React Hooks
- Event handling
- Responsive UI/UX design

# ⚡ Installation Guide

## 🔹 Step 1 — Clone Repository

```bash
git clone https://github.com/yourusername/ai-chatbot-ui.git
```

---

## 🔹 Step 2 — Move to Project Folder

```bash
cd ai-chatbot-ui
```

---

## 🔹 Step 3 — Install Dependencies

```bash
npm install
```

---

## 🔹 Step 4 — Run Project

```bash
npm run dev
```
